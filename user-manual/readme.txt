There are 3 ways to configure the macropad:

1. Using the dedicate EezBotFun Config APP
   This is the easiest way but only windows is supported.

2. Using online website
   Os indepentend, only requirement is that the web browser support USB-Serial. Such as chrome, safari and edge.

3. Hacking
   Write the script then put it under device's scripts folder, that 's all. Refer to "Script Quick Start" .
   Copy the icon you want to the device's app_icons folder to if you want to customise the display. Refer to "User Mannual" for the naming and format of the icon file.
